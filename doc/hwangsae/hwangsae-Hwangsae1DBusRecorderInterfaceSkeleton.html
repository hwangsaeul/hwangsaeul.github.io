void
hwangsae1_dbus_recorder_interface_call_stop
                               (Hwangsae1DBusRecorderInterface *proxy,
                                const gchar *arg_edgeId,
                                GCancellable *cancellable,
                                GAsyncReadyCallback callback,
                                gpointer user_data);

Asynchronously invokes the Stop() D-Bus method on proxy . When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call hwangsae1_dbus_recorder_interface_call_stop_finish() to get the result of the operation.

See hwangsae1_dbus_recorder_interface_call_stop_sync() for the synchronous, blocking version of this method.

Parameters

A Hwangsae1DBusRecorderInterfaceProxy.

Argument to pass with the method invocation.

  

cancellable

A GCancellable or NULL.

[nullable]

A GAsyncReadyCallback to call when the request is satisfied or NULL.

User data to pass to callback .
