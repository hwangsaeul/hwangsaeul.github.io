void
hwangsae1_dbus_edge_interface_call_register
                               (Hwangsae1DBusEdgeInterface *proxy,
                                const gchar *arg_id,
                                guint arg_mode,
                                GCancellable *cancellable,
                                GAsyncReadyCallback callback,
                                gpointer user_data);

Asynchronously invokes the Register() D-Bus method on proxy . When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call hwangsae1_dbus_edge_interface_call_register_finish() to get the result of the operation.

See hwangsae1_dbus_edge_interface_call_register_sync() for the synchronous, blocking version of this method.

cancellable

A GCancellable or NULL.

[nullable]
